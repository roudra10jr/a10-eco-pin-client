EcoPin is a community-driven platform for reporting, tracking, and resolving environmental issues in your local area. The platform allows users to contribute, track, and participate in environmental initiatives while fostering a sense of community responsibility.

## Features

-   **User Authentication:** Secure login and registration with email/password and Google Sign-In.
-   **Issue Management:** Users can report new environmental issues, view all issues, and track their resolution status.
-   **Community Stats:** Displays total registered users, total issues reported, resolved issues, and pending issues.
-   **Dark & Light Mode:** Toggle between dark and light themes seamlessly.
-   **Responsive Design:** Fully responsive layout for desktop, tablet, and mobile devices using Tailwind CSS and DaisyUI.
-   **User Contributions:** Track your reported issues and contributions with personalized dashboards.

## Tech Stack

-   Frontend: React, Tailwind CSS, DaisyUI
-   Backend: Node.js, Express.js
-   Database: MongoDB
-   Authentication: Firebase Authentication
